Project Description

Realestate Services Application

Based on Spring Monolithic Architecture


Introduction

This Real Estate Management Application is designed to streamline and enhance the process of property management, facilitating efficient property listing, user interactions, and administrative tasks. Developed using the Java Spring Boot framework, the application follows a Monolithic architecture for simplicity and ease of maintenance. The comprehensive solution covers modules such as Property Management, User Profiles, Listing Bookings, Transaction Processing, Property Reviews, and Property Ratings.



Technology Stack

Programming Language : Java

Framework : Spring Boot

Architecture : Monolithic

Database : MySQL & MongoDB

Other Tools : Lombok,Swagger,Actuator, RestApi



Modules

Property Management: Manages the list of properties, including details such as location, size, amenities, and pricing.Handles property types, availability, and any associated rules or restrictions.


User Profiles: Manages user information, including profiles, preferences, and historical property data.Provides functionalities for user authentication and authorization.


Listing Bookings: Allows users to search for properties based on criteria such as location, size, and amenities.Facilitates the booking of property listings, including lease duration and optional add-ons.


Property Reviews: Enables users to submit reviews and feedback on their property rental experiences.Manages a repository of user reviews for each property, contributing to the overall property rating.


User Subscription Plan: Manages user subscription details, preferences, and access levels.Offers tiered subscription plans with varying benefits, such as priority access to newly listed properties, exclusive discounts, and personalized recommendations.


Owner Subscription Plan: Facilitates property owners in subscribing to tailored plans for listing and managing their properties.Provides features like enhanced property visibility, analytics on property performance, and dedicated customer support for subscribed owners.


Transaction Processing: Integrates a payment gateway for secure and seamless transaction processing.Supports various payment methods and ensures the confidentiality of user financial information.



Application Health Check Mechanism

The application has a health check mechanism implemented using Spring Boot Actuator. The Actuator module provides useful insights into the Spring environment for a running application with functions for health checking and metrics gathering. The health check endpoint is exposed over HTTP and JMX.



Monitoring and Observability

The monitoring and observability of the application are achieved using health check functions in Spring Boot applications and making them observable by capturing useful health metrics. The application integrates with popular monitoring tools for predicting system health in the future by observing any anomalies in a series of metrics like memory utilization, errors, and disk space.



Payment Processing

Integrates a payment gateway for secure and seamless transaction processing. Supports various payment methods and ensures the confidentiality of user financial information.
