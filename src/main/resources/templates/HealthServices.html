Project Description

Health Services Application

The Health Services Application is designed to provide a seamless and secure experience for users in managing their healthcare needs. With a Microservices Architecture, the project ensures modularity, scalability, and robustness. Users can access features like appointment booking, medicine ordering, and insurance policies through a user-friendly interface. Doctors, on the other hand, can manage appointments, write and read blogs, and monitor their availability slots.

Key features include secure authentication using Spring Security and JWT, circuit breakers for preventing failures, and monitoring performance using Spring Actuator. Data is stored efficiently using MongoDB and MySQL for different services.

The project comprises Microservices such as User Service, Doctor Service, BlogPost Service, Medicine Service, and an API Gateway for centralized access.

Overview:

  • This project showcases the development of a comprehensive Health Services Application built on a robust Microservices Architecture with synchronous communication using REST API. The technology stack includes Java, Spring Boot, Hibernate, MySQL, MongoDB, and employs a total of 7 Microservices to ensure scalability, modularity, and efficiency.
